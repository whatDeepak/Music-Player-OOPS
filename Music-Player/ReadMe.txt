# Music Player Project

## Introduction
This project is a simple music player application developed as part of a Programming assignment or personal project. The music player allows users to manage playlists, play, pause, stop, and skip songs, and includes a visualizer for an enhanced music experience.

## Features
- **Play and Pause:** Control the playback of music with a play/pause button.
- **Stop:** Stop the currently playing music.
- **Volume Control:** Adjust the volume with keyboard shortcuts.
- **Skip Songs:** Navigate to the next song in the playlist.
- **Playlist Management:** Create, load, and save playlists.
- **Visualizer:** Enjoy a visual representation of the music through dynamic bars.

Usage
Use the play/pause button or keyboard shortcut (Space) to control playback.
Adjust volume with the Up and Down arrow keys.
Skip songs with the 'N' key.
Create and manage playlists by following the on-screen instructions.
